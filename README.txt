========================================
    GUEST MANAGEMENT SYSTEM
========================================

🚀 CÁCH SỬ DỤNG:

1. Giải nén file zip này vào một thư mục
2. Double-click vào file "start.bat"
3. Chờ hệ thống cài đặt và khởi động
4. Mở trình duyệt tại: http://localhost:3000

📋 YÊU CẦU HỆ THỐNG:

✅ Python 3.8+ (tự động cài đặt dependencies)
✅ Node.js 16+ (cần cài đặt trước)
✅ Internet connection (để cài đặt packages)

🔧 NẾU GẶP LỖI:

1. Kiểm tra đã cài Node.js chưa: node --version
2. Kiểm tra đã cài Python chưa: python --version
3. Chạy lại file start.bat
4. Kiểm tra port 3000 và 8000 có bị chiếm không

📞 HỖ TRỢ:
- Backend API: http://localhost:8000
- Frontend: http://localhost:3000
- API Docs: http://localhost:8000/docs
